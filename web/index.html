SutraLang Semantic Grammar

SutraLang maps programming operations to Panini's traditional **Karaka-Kriya** semantic structures. In this environment, program states are created and morphed via algebraic operations (Dhatus) rather than Western assignment syntax.

1. Srujana (सृजन) — Creation

Declares a variable state (Karta) with a starting value (Maan).

ek variable counter value 0
counter + sruj(maan=0)

2. Vardhanam (वर्धनम) — Increment

Increases the value of a state variable by an instrument (Karana).

counter ko 5 se badhao
counter + vrdh(karana=5)

3. Hrasanam (ह्रासनम) — Decrement

Decreases the value of a state variable.

counter se 1 se kam karo
counter + hras(karana=1)

4. Web Search (खज् - Search)

Queries the web via DuckDuckGo and stores result in variable.

result ko "Polymarket price" se khojo
result + khaj(karana="Polymarket price")

5. PDF Reader (पठ् - Read)

Extracts semantic keywords from local PDF.

res ko "paper.pdf" aur "data" se padho
res + path(karana="paper.pdf", query="data")

6. Coding Shell (शोध् - Execute)

Executes shell or python script natively.

out ko "python -c 'print(5*5)'" se shodh_karo
out + shodh(karana="python -c 'print(5*5)'")
